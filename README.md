# BlockSiege

<p align="center">
  <img src="images/blocksiege-icon-256.png" alt="BlockSiege Logo" width="220">
</p>

BlockSiege is a Paper plugin that turns block-based privates into a real siege system.

Players can place private blocks with different sizes and durability, manage members, protect territory, and use custom siege TNT to attack enemy bases.

## Features

- Block-based privates
- Multiple private tiers with different sizes and HP
- Unique private block
- Member management
- Private HP display above the block
- Custom siege TNT
- TNT crafting recipes
- Recipe book support
- TNT information menu
- Admin private tools
- Destruction logging
- Configurable settings
- Update checker
- Russian, English, and German language support
